# 필사 — 타이핑으로 읽는 공간

좋아하는 글을 한 글자씩 따라 적으며 천천히 읽어 나가는, 단일 HTML 파일 기반의 필사·타자연습 도구입니다. 빈 줄을 기준으로 글을 장면 단위로 나눠 연습하고, 읽은 만큼 진행이 자동 저장되어 다음에 열면 그 자리에서 이어집니다.

## 사용법

`index.html`을 브라우저로 열면 바로 시작됩니다. GitHub Pages로 배포한 경우 그 주소로 접속하면 됩니다.

- **본문 추가**: 우측 상단 설정(⚙)에서 두 가지 방법으로 글을 넣을 수 있습니다.
  - *본문 붙여넣기*: 제목·작가를 적고 글 본문을 붙여넣으면, 빈 줄(엔터 두 번) 기준으로 장면이 자동으로 나뉩니다.
  - *JSON 불러오기*: `[{ "text": "...", "source": "제목 · 작가" }]` 형식의 JSON을 붙여넣습니다.
- **진행 저장**: 치는 동안 자동으로 저장됩니다. 다시 열면 마지막 장면의 그 자리에서 이어지고, 끝까지 친 장면은 미리보기에 금빛으로 쌓입니다.
- **단축키**: `Enter` 완료 시 다음 장면 · `Esc` 현재 장면 다시 · `Tab` 입력창 포커스
- **초기화**: 설정의 "진행 기록 지우기"로 처음부터 다시 시작할 수 있습니다.

데이터와 진행 기록은 모두 그 브라우저(localStorage)에 저장됩니다. 기기·브라우저마다 따로 보관됩니다.

## 배포 (GitHub Pages)

1. 이 저장소를 GitHub에 올립니다.
2. 저장소 **Settings → Pages → Source**를 `main` 브랜치 / `root`로 설정하고 저장합니다.
3. 몇 분 뒤 `https://<사용자>.github.io/<저장소>/` 주소로 어디서든 접속할 수 있습니다.

## 개인 필사 데이터에 대하여

이 저장소의 기본 데이터(`index.html` 안의 안내 문장)는 직접 작성한 것으로 저작권 문제가 없습니다.

저작권이 있는 책·에세이·칼럼 등 **타인의 글은 이 저장소(특히 공개 저장소)에 올리지 마세요.** 개인적으로 혼자 필사 연습에 쓰는 것은 사적 이용에 해당하지만, 공개된 곳에 올려 두는 것은 복제·전송에 해당해 저작권 문제가 될 수 있습니다.

개인 필사 자료는 `sentences.json`처럼 별도 파일로 본인만 보관했다가, 사용할 기기에서 설정창에 붙여넣어 쓰세요. `.gitignore`가 `sentences.json`이 실수로 올라가는 것을 막아 줍니다.
