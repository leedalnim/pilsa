문장 데이터 관리

글 본문으로 추가

글 본문을 붙여넣으면 문장/줄 단위로 나눠 자동 등록합니다. 맨 앞 # 제목 줄은 작품(탭) 구분이 되고, 한 작품 안에 여러 장(章)도 넣을 수 있습니다.

파일을 올리면 위 칸에 자동으로 채워집니다

내보내기 (백업 · 노션 옮기기)

지금 등록된 모든 글(전체 작품·장면)을 밖으로 빼냅니다.

노션에 붙여넣기 좋은 읽기용 .txt 파일이 데스크탑(다운로드 폴더)에 저장됩니다. → 파일 열어 복사 → 노션에 붙여넣기.
다른 기기·브라우저의 이 앱에 다시 불러올 때 쓰는 형식(클립보드로 복사).

불러오기 (JSON)

글(산문)·.txt 파일을 넣으려면 위쪽 「글 본문으로 추가」에서 「📄 텍스트 파일(.txt) 올리기」를 쓰세요. 여기 JSON 불러오기는 구조화된 데이터를 통째로 교체할 때만 씁니다.

위 「JSON 복사」나 다른 곳에서 받은 JSON을 붙여넣고 불러오기를 누르세요. 이 브라우저에 저장되어 다음에 열어도 유지됩니다.

[ { "text": "긴 문단 텍스트...", "source": "출처 (선택)" }, { "text": "다음 문단...", "source": "" } ]

다른 기기에서 이어하기 (회사 ↔ 집)

지금 친 위치를 코드로 만들어 메모·메신저로 나에게 보내고, 다른 기기에서 붙여넣으면 그 자리에서 이어집니다. 작품 데이터는 기본 제공되니 위치만 옮기면 돼요.
